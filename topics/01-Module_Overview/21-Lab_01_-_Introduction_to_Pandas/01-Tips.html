Tips

In our Pre-Start Lab you looked at the tips dataset using whatever tools that you were familiar with. Today we will revisit the tips dataset using pandas while working within a jupyter-lab interface.

The above file structure has a number of features that we find useful:

  • Structure is based around datasets — which often can be large, so do not want to have multiple versions.
  • The number prefix (01- etc) is useful to ensure desired sort order in file viewer.
  • Within each dataset folder, the folder data contains the original dataset. For most of our datasets this will be just one or two files, but in general this could be arbitrary complicated.
  • The notebooks have naming convention
1
    PREFIX-DATASET_-_TASK.ipynb

where

  • PREFIX is a two-digit, zero-padded prefix to control sort order in file explorer and in jupyter-lab.
  • DATASET is the name of the dataset — this is useful when you have multiple notebooks open across multiple datasets.
  • TASK — the main data mining task that the notebook is focused on.

Step 1 — Download and import dataset

Download the dataset, tips, and place it in your data folder. So following the above suggested structure, place it into folder datasets/01-Tips/data.

Step 2 — Run up a jupyter-lab interface

On Windows (assuming you have anaconda/miniconda) you:

  1. Start anaconda navigator and then Launch jupyter-lab
or (my preference, since I find the navigator to be slow)

  1. Go to the Windows start menu and select Anaconda Prompt under Anaconda3 (see image).
  2. At the prompt enter jupyter-lab
  3. Your browser should open showing interface similar to than shown below

On linux/OS X you:

  1. Open any terminal
  2. At the prompt enter jupyter-lab and a browser should open showing the jupyter-lab interface.
