Module Overview

  • Pre-start lab: Software installation and overview
  • Lecture: Module Introduction

Welcome to Data Mining,
Before we start this module with our first lecture (Friday, 19 Jan) we have a lab session (Monday, 15 Jan) with the aim to ensure that everybody has set up the software required for this module and to introduce the python environment and packages commonly used in data analysis.
The first lecture (19 Jan) will give an overview of the module — what will be covered, how it will be delivered and assessed. In addition, background concepts in statistics and optimisation are discussed.

  • Pre-Start Lab - Software Setup

    The aims of this lab session are:
    • Download and install the Anaconda distribution (Anaconda (python 3.11 64-Bit)).
    • Use the anaconda package manager, conda, to install some additional packages.
    • Create/edit/run python notebooks using the jupyter-lab interface.
    In parallel with above, we want to review your existing data analysis skills with:
    • Data analysis of the Tips dataset using software tools of your choice..

  • Pre-Start Lab

    Typically the Monday lab sessions will based on the material covered in the preceding Friday's lecture. However, for the first week we will run a more informal (and hopefully relaxed) session where you are asked to examine a small dataset using whatever tools you prefer. We want you to focus on the data, and what questions should one ask when given data. Later (next week) we will worry about software and learning new tools (python, pandas, seaborn, scikit-learn, etc). The aims of this lab session are:
    • Review of existing basic data exploration skills.
    • Develop familiarity with Moodle, in particular Moodle activities (Quiz)
