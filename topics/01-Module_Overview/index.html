Welcome to Data Mining,
Before we start this module with our first lecture (Friday, 23 Jan) we have a lab session (Monday, 19 Jan) with the aim to ensure that everybody has set up the software required for this module and to introduce the python environment and packages commonly used in data analysis.
The first lecture (23 Jan) will give an overview of the module — what will be covered, how it will be delivered and assessed. In addition, some background concepts in statistics and optimisation are discussed.

  • Pre-Start Lab - Software Setup

    The aims of this lab session are:
    • Download and install the Anaconda distribution (Anaconda (python 3.13 64-Bit)).
    • Use the anaconda package manager, conda, to install some additional packages.
    • Create/edit/run python notebooks using the jupyter-lab interface.
    In parallel with above, we want to review your existing data analysis skills with:
    • Data analysis of the Tips dataset using software tools of your choice..
