Anomaly Detection (pyod) and Recommender Systems (surprise)

As usual, each notebook should be downloaded to your laptop and run using jupyter-notebook or (preferably) jupyter-lab, as described in previous practicals.

The notebooks can be obtained from the following links:

  • Applying outlier detection from the PYOD library to Digits data
  • Worked example of the SlopeOne recommender being applied to a small dataset
  • Applying recommenders from the Surprise library to FilmTrust data

Installation

You will need to update your python packages by installing pyod and scikit-surprise, using the following commands:

Resources

The datafile used for outlier detection came from the ODDS collection of datasets. The format has been standardised and can be obtained from here:

  • Selected pixel values for '0's (inliers) and '6's (outliers) in the MNIST digits data.

  • Another file that should be added to the data folder for this notebook is this overview of pyod results.

Both files should be downloaded and placed in a data folder below where you placed the anomaly detection notebook.

  • The example dataset for the SlopeOne notebook can be downloaded from here and should be placed in a data folder below where you placed the SlopeOne notebook.

  • The FilmTrust data can be downloaded using the code in the notebook. However, for your convenience it can also be downloaded directly to the data folder from here.
