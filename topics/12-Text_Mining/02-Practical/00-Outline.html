Natural Language Processing: NLTK and sklearn

As usual, each notebook should be downloaded to your laptop and run using jupyter-lab, as described in previous practicals.

The notebooks can be obtained from the following links:

  • 1. Using NLTK to process a variety of documents
  • 1a. NLTK resource downloader for the NLTK_example.ipynb notebook - run this ONCE before running NLTK_example.ipynb
  • 2. Using NLTK to preprocess Amazon reviews and to compare NLTK's VADER bag-of-words Sentiment Analyser with HuggingFace's RoBERTA transformer-based Sentiment Analyser
  • 2a. NLTK resource downloader for the amazonReview-sentiment.ipynb notebook - run this ONCE before running amazonReview-sentiment.ipynb
  • 3. Using sklearn TF-IDF to derive word embedding and hence to compare inaugural speeches
  • 4. Scrape yelp reviews and use HuggingFace BERT transformer to classify the reviews by sentiment levels, comparing them with star ratings chosen by the reviewers

Installation

These notebooks require a large number of dependencies. I had trouble adding these dependencies to my existing conda datamining environment.

Therefore, students are advised to create a conda environment specifically for NLP purposes.

For your convenience, I have included an environment specification file called nlp313.yml that students should download and use as follows:

1
conda env create -f nlp313.yml

This nlp313 environment should be used when running the notebooks. It can be chosen using Anaconda Navigator, or from the command line using

1
conda activate nlp313

The other complexity is that, even when the relevant packages have been installed, there are additional resources that need to be downloaded, from stop-words to trained models, for the code to run.

These resources need to be downloaded just once, so the code that downloads them can be commented out afterwards.

Please run NLTK_example-INIT.ipynb and amazonReview-sentiment-INIT.ipynb before running the corresponding NLTK_example.ipynb and amazonReview-sentiment.ipynb notebooks.

The NLTK resource downloader application provides a means of downloading specific resources directly, or grouped as Collections, of which the all-nltk, book and popular collections probably provide all you need for most text processing tasks.

Resources

The NLTK package comes with optional document corpora, which are used by notebook 1.

Notebook 2 uses Amazon reviews which were obtained from kaggle and was downloaded for convenience.

Notebook 3 uses inaugural speeches which were obtained from kaggle and downloaded for convenience.

Notebook 4 uses web scraping to collect the yelp reviews whose sentiment is then assessed.
