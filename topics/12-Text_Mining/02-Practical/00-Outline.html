Natural Language Processing: NLTK and sklearn

As usual, each notebook should be downloaded to your laptop and run using jupyter-lab, as described in previous practicals.

The notebooks can be obtained from the following links:

  • 1. Using NLTK to process a variety of documents
  • 2. Using sklearn TF-IDF to derive word embedding and hence to compare inaugural speeches
  • 3. Using NLTK to preprocess Amazon reviews and to compare NLTK's VADER bag-of-words Sentiment Analyser with HuggingFace's RoBERTA transformer-based Sentiment Analyser
  • 4. Scrape yelp reviews and use HuggingFace BERT transformer to classify the reviews by sentiment levels, comparing them with star ratings chosen by the reviewers

Resources

The NLTK package comes with optional document corpora, which are used by notebook 1.

Notebook 2 uses inaugural speeches which were obtained from kaggle and downloaded for convenience.

Notebook 3 uses Amazon reviews which were obtained from kaggle and was downloaded for convenience.

Notebook 4 uses web scraping to collect the yelp reviews whose sentiment is then assessed.
