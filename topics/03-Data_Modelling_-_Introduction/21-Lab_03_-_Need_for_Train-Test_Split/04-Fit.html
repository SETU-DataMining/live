Fit / Train Model

To find our model, we use the sklearn model LinearRegression which we import and create an instance of using

1
2
from sklearn.linear_model import LinearRegression
model = LinearRegression()

Note: We (Bernard) will cover the theory of linear regression next week, for now we can just treat this model as a black box, that given numerical inputs predicts numerical output, with the aim of minimising the mean square error (MSE) between the predicted values and given output values.

To fit this model to the train data set we use

1
model.fit(X_train_poly, y_train)

How well does this model work? To use the MSE metric that we discussed in class we import this via

1
from sklearn.metrics import mean_squared_error

and evaluate it by generating predictions and comparing predictions to true values using

1
2
y_train_pred = model.predict(X_train_poly)
mean_squared_error(y_train, y_train_pred)
